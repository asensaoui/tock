Bootloader Attributes
======================

This package helps the Tock build system generate static attributes that are
inserted into the bootloader binary at build time. These attributes are encoded
as static arrays and the linker inserts them into the correct spot in the
binary.

The resulting file that gets included into the board file at compile time looks
like:
